docker使用

docker命令:http://www.runoob.com/docker/docker-command-manual.html
1.启动docker

systemctl enable docker

docker searck 镜像名

dock pull 镜像名
指定版本
docker pull 镜像名:xxx

docker images

docker ps

docker rmi image-id

dock run -p 8080:8080 –name 自定义镜像别名 -d tomcat:latest
-p 指定外部映射容器内部端口
-d 后端运行
-P 随机分配映射端口

docker logs tomcat

docker exec
-d :分离模式: 在后台运行
-i :即使没有附加也保持STDIN 打开
-t :分配一个伪终端

docker commit
-a :提交的镜像作者；
-c :使用Dockerfile指令来创建镜像；
-m :提交时的说明文字；
-p :在commit时，将容器暂停。
